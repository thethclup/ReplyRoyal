# Reply Royale 👑

**Reply Royale** is a fast, funny, and competitive reply battle royale game built on **Base Mainnet**. You are a **Reply Gladiator** fighting in the ultimate social media arena. Craft the funniest, smartest, or most savage replies to climb the leaderboard and become the Reply King. Last player standing or highest score wins the royale.

## Features

- ⚔️ **Rapid-Turn Reply Battles**: Real-time action evaluating trends and comments to select the best reply.
- 🚀 **Hype & Toxicity Mechanics**: Balance your replies to fill your Hype Meter and avoid shadowbans.
- ⛓️ **On-Chain Transactions**:
  - Securely submit your highest Royal Placement or best score directly to the Base blockchain.
  - Features real `SIWE` (Sign-In with Ethereum) capabilities out-of-the-box.
  - "Say GM" on-chain interactions.
- 🤖 **ERC-8004 Trustless Agents**: Native integration for smart agent orchestration on-chain.
- 🧾 **ERC-8021 Transaction Attribution**: Full transaction attribution routing via builder codes.
- 💥 **Abilities & Combos**: Deploy Savage Burns, Ratio Attacks, and Viral Boosts. Achieve massive multiplier combos.
- 💅 **Vibrant Aesthetic**: Neon pink, cyan, and green social arena design running on a smooth mobile-first experience.

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` (Rename `.env.example` to `.env` and fill the variables — *Do not commit sensitive keys*).
4. Run the development environment:
   ```bash
   npm run dev
   ```
